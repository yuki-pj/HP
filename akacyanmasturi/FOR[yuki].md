# 御香宮 赤ちゃんまつり プロジェクト

## 概要
このプロジェクトは、既存の「御香宮 赤ちゃんまつり」Webサイト（https://balancetec.co.jp/project/akachan-matsuri.html）をローカル環境で完全に再現し、検証・修正を行うものです。

## 完了した作業
1.  **HTML修正**:
    -   `index.html` 内の画像の絶対パス（`/data/...`）をローカル相対パス（`media/data/...`）に置換。
    -   プロトコル相対URL（`//...`）を `https://...` に修正し、ローカル環境での読み込みエラーを解消。
    -   ヘッダー部分で欠損していたSVGロゴデータを復元修正。

2.  **リソース補完**:
    -   不足していたファビコン画像群（`media/data/favicons/site/`）をダウンロード・配置。
    -   CSSで参照されていた画像（`media/data/themes/getta/img/bg01.png`, `bg02.png`, `logo.png`）の存在を確認・配置。
        -   ※ `bg01.png` はローカルに存在しますが、現在のページでは使用されていないため表示されませんが、正常です。

3.  **CSS修正**:
    -   `css/import.css` 内の `url(../img/logo.png)` を正しいローカルパス `url(../media/data/themes/getta/img/logo.png)` に修正。

4.  **検証**:
    -   ブラウザ自動テストにより、404エラー（リンク切れ）がないことを確認。
    -   ローカル環境と元サイトの視覚的比較を行い、一致していることを確認。
    -   フッターのロゴや背景画像が正しく表示されていることを確認。

## ディレクトリ構成
- `index.html`: メインページ
- `css/`: スタイルシート
- `js/`: JavaScriptファイル
- `media/`: 画像リソース（元サイトのディレクトリ構造を維持）
