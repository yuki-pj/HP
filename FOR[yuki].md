# 御香宮 -赤ちゃんまつり- プロジェクト概要

## プロジェクトの目的
「御香宮 -赤ちゃんまつり-」のWebサイト制作プロジェクトです。
安産・子育ての神様として知られる御香宮神社でのイベントや、その背後にある「命の尊さ」「地域社会との繋がり」を伝えることを目的としています。

## 技術スタック
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (jQuery, Vanilla JS)

## ディレクトリ構造
- `index.html`: トップページ
- `css/`: スタイルシート
- `js/`: JavaScriptファイル
- `media/`: 画像などのメディアリソース
- `rule/`: デザインやコーディングのルール・ガイドライン

## 主要機能
- ヒーローエリアのアニメーション（実装予定）
- スクロール連動ナビゲーション
- アコーディオンメニュー（スマホ版）
- 助産師インストラクター画像の浮遊アニメーション
