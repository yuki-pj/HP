# プロジェクト概要: 赤ちゃん祭り LP

このプロジェクトは、「御香宮 -赤ちゃん祭り-」のランディングページ（LP）を作成・修正するものです。
安産や子育ての神様として知られる御香宮神社でのイベントや、赤ちゃん回廊などの取り組みを紹介しています。

## 主な機能・構成
- **ヒーローセクション**: アニメーションを用いた印象的なトップ画面。
- **赤ちゃん祭りについて**: イベントの主旨や背景の説明。
- **御香宮神社の紹介**: 神社の歴史や由緒。
- **活動内容**: 妊婦さんや家族向けのサポート活動。
- **体験・プログラム**: 祈願祭、セミナー、回廊などの体験コンテンツ。
- **FAQ**: よくある質問。

## 技術スタック
- HTML5
- CSS3 (SCSS)
- JavaScript (jQuery, GSAP)
