# Antigravity HP Rule Directory

## 概要
このディレクトリ（`rule/`）は、**Antigravity HP** プロジェクトにおける開発・デザイン・運用に関するルールやガイドラインを格納するための場所です。

## 構造
- `visual_adjustments_and_designer_mindset_for_beautiful_ui.md`: 美しいUIを構築するための視覚的調整とデザイナーの思考法についてまとめたドキュメント。
- `color/`: 配色に関するルールや資料を格納するディレクトリ。
    - `web_site_basic_color_rules.md`: **[New]** 世界基準のWebデザイン配色ルール。論理と知識に基づく配色の基礎、黄金比率、実践ワークフローなど。
- `think/`: （思考プロセスやメモなどを格納するディレクトリと推測）

## 目的
プロジェクトの品質を保ち、一貫性のあるデザインとコードを維持するために参照されるべき基準を管理します。
