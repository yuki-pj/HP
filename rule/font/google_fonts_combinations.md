# Google Fontsで再現！ 日本語×英語のフォント組み合わせ 9選

Google Fontsを使用して、日本語と英語のフォントを効果的に組み合わせた9つのデザインパターンを紹介します。

---

## ① 上品・きれいめ（洗練）
**コンセプト:** 私を整える / Elegant Life
- **日本語:** Zen Old Mincho
- **英語:** Pinyon Script
- **特徴:** すっきりとした明朝体と、流れるような筆記体。高級感を出したい時に。

## ② 大人かわいい（愛され）
**コンセプト:** お気に入り / My favorite
- **日本語:** Zen Maru Gothic
- **英語:** WindSong
- **特徴:** 丸みのある文字に、手書き英語を添えて。甘すぎない「大人のかわいさ」です。

## ③ 抜け感・雑誌風（こなれ感）
**コンセプト:** 持たない暮らし / Simple Style
- **日本語:** Noto Sans JP
- **英語:** Cormorant Garamond
- **特徴:** 余白をたっぷり取って、文字は小さめに。ライフスタイル誌のような雰囲気。

## ④ エモい・儚げ（情緒）
**コンセプト:** 春の気配 / Memories
- **日本語:** Zen Kurenaido
- **英語:** Cormorant Garamond
- **特徴:** 繊細な手書き文字が、写真の空気を語ります。思い出やポエム調の投稿に。

## ⑤ 親しみやすさ（安心）
**コンセプト:** はじめまして / Hello!
- **日本語:** M PLUS Rounded 1c
- **英語:** Quicksand
- **特徴:** カドが丸い文字は、読む人にストレスを与えません。自己紹介や、優しい案内に。

## ⑥ 信頼・ビジネス（誠実）
**コンセプト:** 仕事の流儀 / Work Skill
- **日本語:** Noto Sans JP
- **英語:** Lora
- **特徴:** 「ゴシック」で力強く、「セリフ体」で誠実に。読みやすく、説得力が出ます。

## ⑦ カフェ風（癒し）
**コンセプト:** 週末の楽しみ / Coffee Break
- **日本語:** Zen Maru Gothic
- **英語:** Montserrat
- **特徴:** スッキリした英語と合わせると、メニュー表のようなオシャレな空間に。

## ⑧ インパクト（注目）
**コンセプト:** CHECK! / 見逃し厳禁
- **日本語:** Dela Gothic One
- **英語:** Permanent Marker
- **特徴:** どっしり太い文字は、スマホで流し見していても絶対に目が止まります。

## ⑨ レトロポップ（遊び心）
**コンセプト:** レベルアップ / Level Up
- **日本語:** Nico Moji
- **英語:** VT323
- **特徴:** ゲーム画面のような遊び心を。他の人とカブリたくない時に！
