# Yumeori Official Site Project

## プロジェクト概要
クリエイティブチーム「ゆめおり」の公式Webサイト制作プロジェクトです。
「論理（Web）」と「感性（絵本）」という異なる強みを掛け合わせた唯一無二のクリエイティブチームであることを表現するサイトを構築します。

## 目的
- チームのブランディング（論理×感性）
- 実績（Works）の効果的な展示
- チームメンバーのスキル可視化
- お問い合わせへの誘導

## 技術スタック
- HTML5 (セマンティックマークアップ)
- pure CSS3 (アニメーション、レスポンシブ)
- Vanilla JavaScript (インタラクション)

## 主要コンテンツ
1. **Top Page**: キャッチコピー「論理（Web）と感性（絵本）。異なる技術が織りなす、唯一無二のクリエイティブ。」、ビジュアル、CTA。
2. **Works**: 「Marketing / Web制作（論理）」と「Creative / アートワーク（感性）」の2カテゴリで実績を紹介。
3. **Skill & Team**: スキルマップ（レーダーチャート）、チームとしての総合力表現。
4. **Service Flow**: 制作フロー（ヒアリング重視）。

## 進め方
`lp_creation` スキルのワークフローに基づき、企画・構成 → コンテンツ作成 → デザイン → 実装 → 検証 の手順で進めます。
